EyeFoundYou — Full-Stack Lost & Found Platform

A global lost & found web app live at eyefoundyou.com. Finders post items they've found; claimants submit proof of ownership to claim them. Once a finder approves a claim, the app auto-opens a private chat so both sides can coordinate the physical return. The same underlying claim/approve/chat flow is reused for lost-item posts too.

Key Features:

  • Verified proof-of-ownership claim workflow with approve/reject logic and image evidence.
  • Auto-generated private chat between finder and claimant upon claim approval.
  • Secure authentication via JWT and Google OAuth.
  • Full mobile-responsive redesign — hamburger navigation, bottom-sheet modals, touch-optimized tap targets.
  • Dedicated About, Contact, and Footer pages, plus a working contact form.
  • Print-ready launch flyer with an embedded QR code for in-person outreach.

Engineering & Infrastructure:

  • Designed a relational PostgreSQL schema (users, items, claims, chats, messages) with versioned, idempotent migrations.
  • Built RESTful Express APIs for authentication, item posting, claims, and messaging.
  • Built an image upload pipeline with Multer + Cloudinary for item and claim-proof photos.
  • Owned the full deployment lifecycle: custom domain and DNS (Namecheap), CI/CD via git-push auto-deploy on Vercel (frontend) and Railway (backend).
  • Diagnosed and fixed real production bugs (CORS misconfiguration, SPA routing 404s, incorrect claim-proof image URLs) using direct psql access to the live database.

Technologies Used: React (Vite), Tailwind CSS, Node.js/Express, PostgreSQL, JWT, Google OAuth, Multer, Cloudinary, Vercel, Railway, Git.

Impact: This is a live, production web application with real users — not a class project or tutorial clone. It demonstrates end-to-end ownership: architecture, database design, authentication, deployment, and production debugging, the same skills used in a professional engineering role.
